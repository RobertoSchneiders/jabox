Edit Container

The Container is a cargo installation orchestrated by Jabox. It downloads automatically the Application Server (Tomcat by default), and then consequently downloads and installs Jenkins with some predefined plugins, Nexus and Sonar with some predefined plugins.

The configuration of the Containers is located at $HOME/.jabox/config/containers/ where the specific versions of the applications and their plugins can be updated.

Once the Container is started the deployment is taking place at $HOME/.jabox/cargo/CONTAINER_NAME/

The logs of the container are located at $HOME/.jabox/cargo/CONTAINER_NAME.log
